# Cutscenes do jogo The Final Recipe

Essse repositório contem os links públicos das cutscenes usadas no jogo The Final Recipe.

Para acessar essas cutscenes deve-se acessar o seguinte link: https://joao-pedro-queiroz.github.io/TheFinalRecipeCutscenes/, adicionando, após a última `\`, o nome do arquivo `.mp4` que deseja acessar.

Exemplo: https://joao-pedro-queiroz.github.io/TheFinalRecipeCutscenes/cutscene1.mp4
